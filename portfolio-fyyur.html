The Musical Fyyur App

A musical venue and artist booking site

Fyyur is a musical venue and artist booking site that facilitates the discovery and bookings of shows between local performing artists and venues. This site lets you list new artists and venues, discover them, and list shows with artists as a venue owner.

Github Link

With Python and a flask framework. I was able to build out the data models to power the API and the endpoints by connecting to a PostgreSQL database for storing, querying and creating Information about Artist and Venues on Fyyur

The app has the following functions

  • creating new venues, artists, and creating new shows.
  • searching for venues and artists.
  • learning more about a specific artist or venue.
